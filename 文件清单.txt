🚀 部署步骤
─────────────────────────────────────────────────────────────────────────────

1. GitHub 创建仓库
   - 新建仓库名: roamlab
   - 上传以下文件到根目录:
     • index.html
     • admin.html
     • login.html
     • hero-cyber.mp4
     • hero-satellite.mp4
     • hero-connect.mp4

2. Cloudflare Pages 部署
   - 登录 https://dash.cloudflare.com
   - Pages → Connect to Git
   - 选择 roamlab 仓库
   - Build command: [留空]
   - Build output directory: [/]
   - 部署完成！

3. 配置域名（可选）
   - 自定义域名指向你的网站

4. 测试访问
   - 前端: https://your-domain.com
   - 后台: https://your-domain.com/admin.html
   - 登录: admin@roamlab.com / roamlab2026
